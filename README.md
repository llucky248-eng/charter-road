# The Charter Road (Prototype)

A **top-down 2D fantasy trading** prototype.

Current build:
- Free roam in a tile world (procedural tiles, no external assets)
- Two city zones with different rules (Sunspire / Gloomwharf)
- Collision with water/rocks
- HUD shows which city you're in + its tax/inspection/contraband rules

Controls:
- Move: WASD / Arrow keys
- Interact: E (placeholder)

## Deploy
GitHub Pages: Settings → Pages → `main` branch / root.
